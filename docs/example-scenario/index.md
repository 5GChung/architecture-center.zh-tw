# <a name="azure-example-scenarios"></a>Azure 範例案例

這些範例案例是根據真實世界的客戶案例所撰寫。 它們旨在加速並促進您建置自己的使用案例。 每個解決方案都包含：架構圖，用來探索您可以代入的一些替代方案，以滿足本身的商務需求及解決方案的粗估價格。  大部分的解決方案也會包含一種方法，可在您自己的 Azure 帳戶中輕鬆部署解決方案。

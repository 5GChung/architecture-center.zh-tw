# <a name="advanced-analytics"></a><span data-ttu-id="b5dab-102">進階分析</span><span class="sxs-lookup"><span data-stu-id="b5dab-102">Advanced analytics</span></span>

<span data-ttu-id="b5dab-103">進階分析超越傳統商業智慧 (BI) 的歷程記錄報告與資料彙總，並使用數學、機率及統計模型技術來啟用預測處理和自動化決策制定。</span><span class="sxs-lookup"><span data-stu-id="b5dab-103">Advanced analytics goes beyond the historical reporting and data aggregation of traditional business intelligence (BI), and uses mathematical, probabilistic, and statistical modeling techniques to enable predictive processing and automated decision making.</span></span>

<span data-ttu-id="b5dab-104">進階分析解決方案通常涉及下列工作負載：</span><span class="sxs-lookup"><span data-stu-id="b5dab-104">Advanced analytics solutions typically involve the following workloads:</span></span>

- <span data-ttu-id="b5dab-105">互動式資料探索和視覺效果</span><span class="sxs-lookup"><span data-stu-id="b5dab-105">Interactive data exploration and visualization</span></span>
- <span data-ttu-id="b5dab-106">機器學習模型訓練</span><span class="sxs-lookup"><span data-stu-id="b5dab-106">Machine Learning model training</span></span>
- <span data-ttu-id="b5dab-107">即時或批次預測處理</span><span class="sxs-lookup"><span data-stu-id="b5dab-107">Real-time or batch predictive processing</span></span>

<span data-ttu-id="b5dab-108">大部分進階分析架構包括下列部分或所有元件：</span><span class="sxs-lookup"><span data-stu-id="b5dab-108">Most advanced analytics architectures include some or all of the following components:</span></span>

- <span data-ttu-id="b5dab-109">**資料儲存體**。</span><span class="sxs-lookup"><span data-stu-id="b5dab-109">**Data storage**.</span></span> <span data-ttu-id="b5dab-110">進階分析解決方案需要資料來訓練機器學習模型。</span><span class="sxs-lookup"><span data-stu-id="b5dab-110">Advanced analytics solutions require data to train machine learning models.</span></span> <span data-ttu-id="b5dab-111">資料科學家通常需要探索資料以識別其預測功能，以及資料與其預測值 (稱為標籤) 之間的統計關聯性。</span><span class="sxs-lookup"><span data-stu-id="b5dab-111">Data scientists typically need to explore the data to identify its predictive features and the statistical relationships between them and the values they predict (known as a label).</span></span> <span data-ttu-id="b5dab-112">預測的標籤可以是量化值，例如未來某些事物得財務價值，或班機誤點的持續時間 (以分鐘為單位)。</span><span class="sxs-lookup"><span data-stu-id="b5dab-112">The predicted label can be a quantitative value, like the financial value of something in the future or the duration of a flight delay in minutes.</span></span> <span data-ttu-id="b5dab-113">或可能代表分類的類別 (如「true」或「false」、「班機誤點」或「沒有班機誤點」)，或「低風險」、「中風險」或「高風險」等類別。</span><span class="sxs-lookup"><span data-stu-id="b5dab-113">Or it might represent a categorical class, like "true" or "false," "flight delay" or "no flight delay," or categories like "low risk," "medium risk," or "high risk."</span></span>

- <span data-ttu-id="b5dab-114">**批次處理**。</span><span class="sxs-lookup"><span data-stu-id="b5dab-114">**Batch processing**.</span></span> <span data-ttu-id="b5dab-115">若要訓練機器學習模型，您通常需要處理大量訓練資料。</span><span class="sxs-lookup"><span data-stu-id="b5dab-115">To train a machine learning model, you typically need to process a large volume of training data.</span></span> <span data-ttu-id="b5dab-116">訓練模型可能需要一些時間 (大約幾分鐘到幾小時）。</span><span class="sxs-lookup"><span data-stu-id="b5dab-116">Training the model can take some time (on the order of minutes to hours).</span></span> <span data-ttu-id="b5dab-117">此訓練可使用 Python 或 R 等語言所撰寫的指令碼執行，而且可使用分散式處理平台 (像是 HDInsight 或 Docker 容器中裝載的 Apache Spark) 來相應放大以減少訓練時間。</span><span class="sxs-lookup"><span data-stu-id="b5dab-117">This training can be performed using scripts written in languages such as Python or R, and can be scaled out to reduce training time using distributed processing platforms like Apache Spark hosted in HDInsight or a Docker container.</span></span>

- <span data-ttu-id="b5dab-118">**即時訊息擷取**。</span><span class="sxs-lookup"><span data-stu-id="b5dab-118">**Real-time message ingestion**.</span></span> <span data-ttu-id="b5dab-119">在生產環境中，許多進階分析會將即時資料流饋送到已發佈為 Web 服務的預測模型。</span><span class="sxs-lookup"><span data-stu-id="b5dab-119">In production, many advanced analytics feed real-time data streams to a predictive model that has been published as a web service.</span></span> <span data-ttu-id="b5dab-120">系統通常會在某種形式的佇列中擷取內送資料流，且串流處理引擎會從此佇列中提取資料，並幾近即時地將預測套用到輸入資料。</span><span class="sxs-lookup"><span data-stu-id="b5dab-120">The incoming data stream is typically captured in some form of queue and a stream processing engine pulls the data from this queue and applies the prediction to the input data in near real time.</span></span>

- <span data-ttu-id="b5dab-121">**串流處理**。</span><span class="sxs-lookup"><span data-stu-id="b5dab-121">**Stream processing**.</span></span> <span data-ttu-id="b5dab-122">一旦具有已訓練的模型，指定功能集通常可極快速 (大約毫秒為單位) 執行預測 (或計分) 作業。</span><span class="sxs-lookup"><span data-stu-id="b5dab-122">Once you have a trained model, prediction (or scoring) is typically a very fast operation (on the order of milliseconds) for a given set of features.</span></span> <span data-ttu-id="b5dab-123">擷取即時訊息後，相關功能值就可傳遞至預測服務以產生預測的標籤。</span><span class="sxs-lookup"><span data-stu-id="b5dab-123">After capturing real-time messages, the relevant feature values can be passed to the predictive service to generate a predicted label.</span></span>

- <span data-ttu-id="b5dab-124">**分析資料存放區**。</span><span class="sxs-lookup"><span data-stu-id="b5dab-124">**Analytical data store**.</span></span> <span data-ttu-id="b5dab-125">在某些情況下，預測標籤值會寫入分析資料存放區，以便建立報告及進一步分析。</span><span class="sxs-lookup"><span data-stu-id="b5dab-125">In some cases, the predicted label values are written to the analytical data store for reporting and future analysis.</span></span>

- <span data-ttu-id="b5dab-126">**分析和報告**。</span><span class="sxs-lookup"><span data-stu-id="b5dab-126">**Analysis and reporting**.</span></span> <span data-ttu-id="b5dab-127">正如其名，進階分析解決方案通常會產生某種類型的報表或分析摘要，包含預測的資料值。</span><span class="sxs-lookup"><span data-stu-id="b5dab-127">As the name suggests, advanced analytics solutions usually produce some sort of report or analytical feed that includes predicted data values.</span></span> <span data-ttu-id="b5dab-128">預測的標籤值通常可用來填入即時儀表板。</span><span class="sxs-lookup"><span data-stu-id="b5dab-128">Often, predicted label values are used to populate real-time dashboards.</span></span>

- <span data-ttu-id="b5dab-129">**協調流程**。</span><span class="sxs-lookup"><span data-stu-id="b5dab-129">**Orchestration**.</span></span> <span data-ttu-id="b5dab-130">雖然初始資料探索和模型建立是由資料科學家以互動方式執行，但許多進階分析解決方案會定期以新資料重新訓練模型 &mdash; 持續精細化模型的精確度。</span><span class="sxs-lookup"><span data-stu-id="b5dab-130">Although the initial data exploration and modeling is performed interactively by data scientists, many advanced analytics solutions periodically re-train models with new data &mdash; continually refining the accuracy of the models.</span></span> <span data-ttu-id="b5dab-131">此重新訓練作業可使用協調的工作流程來加以自動化。</span><span class="sxs-lookup"><span data-stu-id="b5dab-131">This retraining can be automated using an orchestrated workflow.</span></span>

## <a name="machine-learning"></a><span data-ttu-id="b5dab-132">機器學習服務</span><span class="sxs-lookup"><span data-stu-id="b5dab-132">Machine learning</span></span>

<span data-ttu-id="b5dab-133">機器學習服務是用來訓練預測模型的數學模型技術。</span><span class="sxs-lookup"><span data-stu-id="b5dab-133">Machine learning is a mathematical modeling technique used to train a predictive model.</span></span> <span data-ttu-id="b5dab-134">一般原則是將統計演算法套用至大型歷程記錄資料的資料集，從而找出其內含欄位之間的關聯性。</span><span class="sxs-lookup"><span data-stu-id="b5dab-134">The general principle is to apply a statistical algorithm to a large dataset of historical data to uncover relationships between the fields it contains.</span></span>

<span data-ttu-id="b5dab-135">機器學習模型通常是由資料科學家執行，他們需要先徹底探索並準備資料再訓練模型。</span><span class="sxs-lookup"><span data-stu-id="b5dab-135">Machine learning modeling is usually performed by data scientists, who need to thoroughly explore and prepare the data before training a model.</span></span> <span data-ttu-id="b5dab-136">此探索及準備通常牽涉到大量的互動式資料分析和視覺效果 &mdash; 通常是在特地針對此工作所設計的互動式工具和環境中，使用 Python 和 R 等語言。</span><span class="sxs-lookup"><span data-stu-id="b5dab-136">This exploration and preparation typically involves a great deal of interactive data analysis and visualization &mdash; usually using languages such as Python and R in interactive tools and environments that are specifically designed for this task.</span></span>

<span data-ttu-id="b5dab-137">在某些情況下，您可以使用[預先訓練模型](/machine-learning-server/install/microsoftml-install-pretrained-models)，其隨附 Microsoft 所取得和開發的訓練資料。</span><span class="sxs-lookup"><span data-stu-id="b5dab-137">In some cases, you may be able to use [pretrained models](/machine-learning-server/install/microsoftml-install-pretrained-models) that come with training data obtained and developed by Microsoft.</span></span> <span data-ttu-id="b5dab-138">預先訓練模型的優點是可立即計分和分類新內容 (即使您沒有必要的訓練資料、用來管理大型資料集或用來訓練複雜模型的資源)。</span><span class="sxs-lookup"><span data-stu-id="b5dab-138">The advantage of pretrained models is that you can score and classify new content right away, even if you don't have the necessary training data, the resources to manage large datasets or to train complex models.</span></span>

<span data-ttu-id="b5dab-139">機器學習有兩大類：</span><span class="sxs-lookup"><span data-stu-id="b5dab-139">There are two broad categories of machine learning:</span></span>

- <span data-ttu-id="b5dab-140">**監督式學習**。</span><span class="sxs-lookup"><span data-stu-id="b5dab-140">**Supervised learning**.</span></span> <span data-ttu-id="b5dab-141">監督式學習是機器學習最常採用的方法。</span><span class="sxs-lookup"><span data-stu-id="b5dab-141">Supervised learning is the most common approach taken by machine learning.</span></span> <span data-ttu-id="b5dab-142">在監督式學習模型中，來源資料包含一組「功能」資料欄位，其與一或多個「標籤」資料欄位之間存在數學關聯性。</span><span class="sxs-lookup"><span data-stu-id="b5dab-142">In a supervised learning model, the source data consists of a set of *feature* data fields that have a mathematical relationship with one or more *label* data fields.</span></span> <span data-ttu-id="b5dab-143">在機器學習程序的訓練階段中，資料集包含功能和已知標籤，並且會套用演算法以符合函式，從而在功能上作業以計算對應的標籤預測功能。</span><span class="sxs-lookup"><span data-stu-id="b5dab-143">During the training phase of the machine learning process, the data set includes both features and known labels, and an algorithm is applied to fit a function that operates on the features to calculate the corresponding label predictions.</span></span> <span data-ttu-id="b5dab-144">一般會保留訓練資料集的子集，並用來驗證訓練模型的效能。</span><span class="sxs-lookup"><span data-stu-id="b5dab-144">Typically, a subset of the training dataset is held back and used to validate the performance of the trained model.</span></span> <span data-ttu-id="b5dab-145">模型經過訓練後，它可以部署到生產環境，以及用來預測未知的值。</span><span class="sxs-lookup"><span data-stu-id="b5dab-145">Once the model has been trained, it can be deployed into production, and used to predict unknown values.</span></span>

- <span data-ttu-id="b5dab-146">**非監督式學習**。</span><span class="sxs-lookup"><span data-stu-id="b5dab-146">**Unsupervised learning**.</span></span> <span data-ttu-id="b5dab-147">在非監督式學習模型中，訓練資料不包含已知的標籤值。</span><span class="sxs-lookup"><span data-stu-id="b5dab-147">In an unsupervised learning model, the training data does not include known label values.</span></span> <span data-ttu-id="b5dab-148">相反地，此演算法會在第一次接觸到資料時就進行預測。</span><span class="sxs-lookup"><span data-stu-id="b5dab-148">Instead, the algorithm makes its predictions based on its first exposure to the data.</span></span> <span data-ttu-id="b5dab-149">最常見的非監督式學習形式是「群集」，其中的演算法會判斷最佳的方法，根據功能中的統計相似性將資料分割成指定的叢集數目。</span><span class="sxs-lookup"><span data-stu-id="b5dab-149">The most common form of unsupervised learning is *clustering*, where the algorithm determines the best way to split the data into a specified number of clusters based on statistical similarities in the features.</span></span> <span data-ttu-id="b5dab-150">在群集中，預測的結果是輸入功能所屬的叢集數目。</span><span class="sxs-lookup"><span data-stu-id="b5dab-150">In clustering, the predicted outcome is the cluster number to which the input features belong.</span></span> <span data-ttu-id="b5dab-151">雖然有時可直接使用它們來產生有用的預測 (例如使用群集來識別客戶資料庫中的使用者群組)，但非監督式學習方法更常用來識別哪些資料最有用，以便在訓練模型時將其提供給監督式學習演算法。</span><span class="sxs-lookup"><span data-stu-id="b5dab-151">While they can sometimes be used directly to generate useful predictions, such as using clustering to identify groups of users in a database of customers, unsupervised learning approaches are more often used to identify which data is most useful to provide to a supervised learning algorithm in training a model.</span></span>
